# 🏏 Cricket RAG Chatbot

A Retrieval-Augmented Generation (RAG) chatbot that answers questions about cricket using a custom knowledge base and Claude AI.

🔗 **Live Demo:** [https://huggingface.co/spaces/ajay232/rag-cricket-chatbot](https://huggingface.co/spaces/ajay232/rag-cricket-chatbot)

---

## 📌 Overview

This chatbot combines a scraped cricket knowledge base with Claude's language understanding to answer questions about players, rules, tournaments, and records. It uses semantic search to find relevant context before generating answers.

---

## 🛠️ Tech Stack

| Component | Technology |
|---|---|
| Language Model | Anthropic Claude (claude-sonnet-4-5) |
| Embeddings | SentenceTransformers (all-MiniLM-L6-v2) |
| Vector Database | ChromaDB |
| Web UI | Gradio |
| Deployment | Hugging Face Spaces |
| Web Search | Anthropic Web Search Tool |
